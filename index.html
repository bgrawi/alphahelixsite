About Us

Phoenix Bioinformatics is a nonprofit organization established in 2013 that aims to ensure scientific data sustainability. They created a web portal called The Arabidopsis Information Resource (TAIR), which maintains a database of genetic and molecular biology data for the model plant Arabidopsis thaliana. TOAST is an existing database and web portal that allows members of the scientific community to contribute Arabidopsis gene function annotations to TAIR.

Team Alpha Helix's goal is to replace the current web portal with a new system, the Genetic Online Annotation Tool (GOAT). GOAT aims to accelerate the process of biocuration and handle plant genomes other than Arabidopsis, supporting annotation of gene functions for any organism with its gene list in the database. This system should allow registered users to submit annotations to ontology terms based on their research, allow curators to review annotation submissions, and allow researchers to view and search for genetic annotations. GOAT will feature greater efficiency and improved usability over the existing system.
